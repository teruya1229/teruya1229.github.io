# BCサービス LP - Cursor Context

## What this repo is
This repository is the landing page (LP) / mini-site for "BCサービス" (Okinawa South).
Primary conversion goal is to drive LINE inquiries and bookings.

## Service / Brand
- Service: Air conditioner cleaning, deep cleaning (完全分解), washing machine cleaning, house cleaning
- Brand axis: 「安心で勝つ」 (Win with reassurance/trust)
- Tone: clean, trustworthy, local, friendly (avoid aggressive salesy language)

## Target area (must keep consistent)
南城市 / 八重瀬町 / 豊見城市 / 糸満市 / 那覇市 / 南風原町 / 与那原町

## Primary KPI priorities
1) CTA click rate to LINE
2) Mobile readability and layout stability
3) Local SEO visibility
4) Page speed (avoid heavy scripts/assets)

## CTA rule of thumb
- Prefer multiple CTAs, especially after:
  - proof sections (before/after, reviews)
  - pricing/service explanation
  - FAQ / anxiety removal
- CTA wording preference: 「LINEで無料相談👇」
- Make CTA visually strong but not “cheap ad” style.

## UX/UI principles for edits
- Mobile-first
- Center alignment as default
- Maintain consistent spacing and typography
- Keep layout simple and fast
- Minimal, safe changes (do not break existing design)

## SEO principles for edits
- Semantic HTML
- Logical heading hierarchy (H1→H2→H3)
- Image alt attributes (local + service intent)
- Internal links should support local pages and service pages (if exist)
- Avoid duplicate titles/headers across pages

## “Do not do”
- Don’t rename/alter confirmed official URLs (LINE, forms, etc.)
- Don’t introduce heavy animation libraries
- Don’t rewrite the whole design unless explicitly asked
