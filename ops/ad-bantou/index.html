広告番頭

Google広告の費用・クリック・CTA・問い合わせ・成約を整理して、継続 / 改善 / 停止を判断する画面

内部確認用ページ（noindex）｜顧客名・住所・電話番号などの個人情報は入力しないでください

Claude in Chrome 作業手順（毎日ここから始める）
  1. Google広告を開く
  2. 期間を「昨日」にする
  3. キャンペーン別に「費用・表示回数・クリック数・平均クリック単価」を確認する
  4. GA4 または LP計測画面で「CTAクリック数」を確認する
  5. LINE・電話・Airリザーブ・Googleカレンダーから「問い合わせ数・成約数」を確認する
  6. 下の「今日の入力」フォームに上から順に入力する
  7. 判定が「停止候補」または「改善」なら、メモに理由を書く（個人情報は書かない）
  8. 最後に「日別ログ」と「LP別集計」を確認する

広告開始前チェック

Google広告を開始する前に、LPのCTA計測が正常か確認するためのチェック欄です。

Claude in Chrome 作業手順（広告開始前チェック）
  1. debug確認URLを開く
  2. CTAをクリックする
  3. 右下に「計測OK」が表示されたらOKにチェック
  4. 実際にLINE送信・電話発信・予約確定はしない
  5. 確認結果をこの広告番頭に保存する
