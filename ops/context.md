## 6. CTA / Copy Rules
CTA文言の基本:
- 第一推奨: LINEで無料相談👇
- 相談起点の文言を優先し、予約は補足で伝える
- CTAは主要セクション後に配置（証拠/価格/不安解消の直後）

コピー原則:
- 短く、具体的、安心重視
- 追加料金なし・相談のみOK・返信目安など不安解消を明記
- 地域密着（沖縄南部）と実績の一貫訴求

## 7. QA Checklist
公開前チェック:
- [ ] 主要リンクが正しい（LINE / フォーム / 内部リンク）
- [ ] CTAクリック導線が全て動作する
- [ ] GA4 debug確認（`?debug=1` でCTAクリック後「計測OK」表示）
- [ ] スマホ表示で崩れがない（FV/CTA/カード/余白）
- [ ] PC表示で崩れがない
- [ ] ページ体感速度が遅くなっていない
- [ ] 見出し階層・alt・基本SEOを維持
